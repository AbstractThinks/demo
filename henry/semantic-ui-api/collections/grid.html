种类

网格（栅格）系统

栅格是用来分隔内容区域的，内容区域之间有一定的间隙。说得更明白一点，栅格系统就是用来做网页布局的！

所有栅格列都可以随意指定宽度（占总列数的百分比或者小于16的数字），但是所有列的宽度的总合不能大于最大值。Semantic UI默认为16列，这很重要，在你设计网页布局的时候请记住所有列的总合要为16列哦。

页面栅格（网格）系统

页面栅格（网格）系统主要是用来做网页的大体布局的，比如你可以把一个页面分成左右两部分，左边占一半的宽度，右边占一半的宽度。当然，你完全可以按照自己的意愿来组织网页布局，而不必拘泥于本文档。最主要的是，你需要明白栅格的用法即可。

Semantic UI会默认在页面栅格（网格）列之间增加250像素的间隙。
