图像控件       Semantic UI学习交流群 262577550

图像控件用于显示一个图像（图片），并加上其它美化效果。

点击切换到定义模式
图像控件种类 图像控件状态 变化的图像控件 图像控件组

图像控件种类

图像

显示一张图片。

默认情况下，图像控件占用和原始图片同样的大小。如果指定图像控件的大小，图片将会被拉伸。

图像链接

使用图片做的链接。

图像控件状态

禁用

处于禁用状态的图像控件可以被显示，但是不能被选择。

变化的图像控件

头像图像控件

图像控件可以被内联的显示成一个头像的样式。

狗狗

圆形图像控件

图像被显示成圆形的形状。

圆角图像控件

图像控件的四角被修饰成圆角形状。

浮动的图像控件

图像控件可以被显示在内容的左边或者右边。
