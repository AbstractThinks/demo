事件

可以通过Javascript代码访问的事件：

$('.ui.accordion').accordion('behavior', argumentOne, argumentTwo...);
open (index) 展开索引为index的内容。
close (index) 关闭索引为index的内容。
toggle (index) 切换索引为index的内容。

手风琴模块的配置
配置可以修改手风琴模块的行为。

配置 默认值 描述
exclusive true 设置为false将允许多个区别同时展开。
collapsible true 设置为false将会一直有一个区域处于展开状态。
duration 500 展开的动画时长。
easing easeInOutQuint EaseInOutQuint is included with accordion, for additional options consider jQuery easing)

回调
手风琴模块的回调方法。

方法 上下文 描述
onOpen active content 展开时的回调。
onClose active content 关闭时的回调。
onChange active content 展开关闭时都会触发的回调。
