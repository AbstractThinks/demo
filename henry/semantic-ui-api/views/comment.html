评论控件       Semantic UI学习交流群 262577550

评论控件主要用来给用户显示一个评论框（或者评论的区域），这个评论框里面包括了用户对网站某个页面上的文章或者视频或者其它内容的评论信息。比如说当前最为流行的友言、多说等社会化评论组件都是评论控件的延伸。
